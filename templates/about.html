About Sorting Office

A comprehensive mail server administration tool for managing domains, users, aliases, and backups.

What is Sorting Office?

Sorting Office is a web-based administration interface for managing a complete mail server setup. It provides an intuitive way to manage domains, users, email aliases, and backup configurations for a Postfix and Dovecot mail server environment.

Built with Rust and modern web technologies, Sorting Office offers a secure, fast, and reliable way to administer your mail server without needing to manually edit configuration files.

Key Features

Domain Management

Add, edit, and manage mail domains with ease

User Management

Create and manage email users and accounts

Alias Management

Set up email aliases and forwarding rules

Backup Configuration

Configure and manage backup MX servers

Statistics Dashboard

Monitor your mail server usage and statistics

Dark Mode Support

Modern UI with dark mode for comfortable use
